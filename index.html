CODAP 실습 가이드

CODAP 실습 가이드: World Bank 데이터 분석 단계별 안내

대상: 고등학교 1학년 | 도구: CODAP | 데이터 파일: worldbank_gdp_lifeexpectancy.csv

1. CODAP 접속 및 프로젝트 생성 (1분)

  1. 웹 브라우저에서 CODAP에 접속합니다.
  2. 상단 메뉴에서 ‘New Document’를 클릭해 새 프로젝트를 만듭니다.

2. 데이터 불러오기 (3분)

  1. 왼쪽 상단의 ‘+ Add Data’ 아이콘을 클릭합니다.
  2. ‘From File…’을 선택해 CSV 파일을 업로드합니다.
  3. 데이터 테이블이 나타나면 변수 목록을 확인합니다.

3. 데이터 전처리: 변수 필터링 및 숨기기 (4분)

  1. 테이블 옵션(톱니바퀴)에서 ‘Hide Columns’로 불필요 칼럼을 숨깁니다.
  2. ‘Filter Cases’로 특정 연도/국가만 남길 수 있습니다.

4. 산점도 만들기 (10분)

  1. 상단 도구 모음에서 Scatter Plot 선택
  2. X축에 GDP per capita, Y축에 Life expectancy 설정
  3. Country를 Color by, Label에 추가
  4. 디스플레이 설정에서 축 범위와 점 크기 조정

5. 막대그래프 만들기 (8분)

  1. Bar Chart 선택
  2. X축에 Country, Y축에 Life expectancy 설정
  3. 평균선 추가 및 내림차순 정렬

6. 인사이트 탐색 및 노트 작성 (7분)

  • 산점도에서 상관 관계 파악
  • 막대그래프에서 국가별 차이 관찰
  • 파인만 기법으로 노트 작성

7. 그래프 내보내기 및 발표 자료 활용 (5분)

  1. Share → Export Image로 PNG 저장
  2. 발표 자료에 이미지 삽입

8. 과제 안내 (2분)

  • 새 데이터(문맹률/평균 교육 연수)로 동일 실습
  • 분석 결과 3문장 요약 제출
